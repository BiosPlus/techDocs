# Scripts & Automation

Collection of scripts, configurations, and automation tools for managing Google Workspace, macOS, and remote operations.

## Overview

This section contains practical scripts and configurations extracted from the [Device-Management repository](https://github.com/BiosPlus/Device-Management) with documentation and usage examples.

## Categories

### [macOS Management](/scripts/macos/)
- Application configuration files (.mobileconfig)
- Custom shell commands and functions  
- System administration utilities
- Device management scripts

### [Google Workspace](/scripts/google-workspace/)
- Chrome extension policies and configurations
- Browser management settings
- Workspace automation tools

## Important Notes

::: warning OSQuery Recommendation
For future scripting implementations, consider deploying OSQuery as recommended in the source repository for better cross-platform compatibility and monitoring.
:::

::: info Jamf Compatibility
`.mobileconfig` files in this documentation are not necessarily Jamf-ready and may require modification for use with Jamf Pro.
:::
